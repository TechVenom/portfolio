---

## 🏛️ Architecture & Design Philosophy

The project follows a **"Zero-Color, High-Motion"** philosophy. By removing the distraction of color, the focus is shifted entirely toward **Typography**, **Geometry**, and **Dynamic Motion**.

### Key Design Pillars:
- **Elite Noir Aesthetic**: A curated monochrome palette (`#000000` to `#FFFFFF`) with high-contrast hierarchy.
- **3D Interaction**: Premium 3D perspective tilt effects using `framer-motion` and `useMotionValue`.
- **Micro-Animations**: Staggered reveals, spring-based progress indicators, and custom cursor spotlight effects.
- **Human-Centric Copy**: Professional, direct, and outcome-focused content tailored for high-level technical recruitment.

---

## 🚀 Core Features

### 🛠️ Technical Skill Matrix
- **3D Skill Cards**: Interactive cards that respond to mouse proximity.
- **Animated Stats**: Spring-physics counters for skill percentages.
- **Dynamic Tabs**: Seamless shared-layout transitions between skill categories.

### 📁 Featured Projects
- **Interactive Modals**: Deep-dive project views with staggered content entrance.
- **Spotlight Effects**: Radial gradient tracking that follows the cursor into project cards.
- **Live Links**: Direct integration with GitHub and production deployments.

### ⏳ Experience Timeline
- **Animated Pathfinding**: Vertical timeline line that draws itself based on scroll position.
- **Active Pulse**: Real-time pulsing indicators for current professional roles.
- **Status Badging**: Clean categorization for Experience, Education, and Certifications.

### 📄 Professional Resume System
- **Automated Generation**: Custom Puppeteer-based pipeline to generate a matching Noir PDF resume from HTML templates.
- **Integrated Download**: Single-click access to a professional, recruiter-ready PDF.

## 🛠️ Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/TechVenom/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Generate Resume PDF**
   *(Note: Requires Puppeteer installed locally)*
   ```bash
   node scripts/generate-resume.js
   ```
